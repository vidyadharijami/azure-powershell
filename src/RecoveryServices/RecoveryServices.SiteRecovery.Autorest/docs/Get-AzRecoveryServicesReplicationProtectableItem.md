# Get-AzRecoveryServicesReplicationProtectableItem

## SYNOPSIS
The operation to get the details of a protectable item.

## SYNTAX

### List (Default)
```
Get-AzRecoveryServicesReplicationProtectableItem -FabricName <String> -ProtectionContainerName <String>
 -ResourceGroupName <String> -ResourceName <String> [-SubscriptionId <String[]>] [-Filter <String>]
 [-SkipToken <String>] [-Take <String>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### Get
```
Get-AzRecoveryServicesReplicationProtectableItem -FabricName <String> -ProtectableItemName <String>
 -ProtectionContainerName <String> -ResourceGroupName <String> -ResourceName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```
